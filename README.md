# Vanguard Skygen

> A feature-rich skyblock/skygen addon for Minecraft Bedrock Edition

**Originally Created by Vyse** | Maintained by AstoriaNova

## ✨ Features

- 💰 **Currency System** - Complete economy management for Bedrock
- 🔄 **Transfer System** - Player-to-player currency transfers
- 🎲 **Gambling System** - Risk vs reward mechanics
- 🧱 **Plot System** - Claim and manage island plots
- 🏪 **Shop Systems** - Buy and sell items with in-game currency
- ⚡ **Script API Integration** - Built with Minecraft Bedrock's Script API

## 📖 About

**Vanguard Skygen** was originally created by **Vyse** and is now maintained by **AstoriaNova**.

This addon has been made **open-source** for the Bedrock community to:
- Learn how to build similar systems using Minecraft's Script API
- Host Vanguard Skygen on your own Bedrock server
- Study the implementation of plot, currency, gambling, and shop mechanics
- Modify and adapt the code for your needs
